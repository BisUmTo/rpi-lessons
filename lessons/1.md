# Lezione 1: Configurazione Iniziale Raspberry Pi

In questa lezione vedremo come preparare la scheda SD, avviare il Raspberry Pi per la prima volta, connettersi via SSH e configurare il nome del dispositivo.

## 1. Preparazione della Scheda SD

1. **Inserimento**: Inserisci la scheda Micro SD con il suo adattatore nel tuo PC.
2. **Accesso alla partizione Boot**: Apri la partizione denominata `boot` che apparirà tra le unità disco.
3. **Abilita SSH**: 
   - Assicurati che il tuo sistema mostri le estensioni dei file.
   - Crea un file vuoto denominato semplicemente `ssh` (senza estensione) nella root della partizione `boot`. Questo permetterà l'accesso remoto al primo avvio.
4. **Espulsione**: Espelli la scheda SD in modo sicuro dal PC.

## 2. Primo Avvio e Collegamento Hardware

1. **Inserimento SD**: Inserisci la Micro SD nel Raspberry Pi.
2. **Rete**: Collega il cavo Ethernet al Raspberry Pi e al router/switch della rete locale.
3. **Alimentazione**: Collega l'alimentatore al Raspberry Pi per accenderlo.
4. **Attesa**: Attendi circa un minuto affinché il sistema operativo si avvii completamente.

## 3. Identificazione dell'Indirizzo IP

Per connetterti al Raspberry Pi, devi conoscere il suo indirizzo IP sulla rete locale.

1. **Download Scanner**: Scarica [Advanced IP Scanner](https://www.advanced-ip-scanner.com/it/download/).
2. **Esecuzione**: Avvia il programma (puoi scegliere la versione portatile selezionando "Esegui").
3. **Scansione**: Clicca su **Scansiona**.
4. **Individuazione**: Cerca nella lista un dispositivo indicato come `Raspberry Pi Foundation`. Segnati l'indirizzo IP corrispondente.

NOTA BENE: se appaiono più indirizzi IP, prova a connetterti (passaggio 4.) a ciascuno di essi finché non trovi quello giusto.

## 4. Connessione Remota (SSH)

1. Apri il terminale (o CMD su Windows).
2. Esegui il comando di connessione:
   ```bash
   ssh pi@indirizzoIP
   ```
   *(Sostituisci `indirizzoIP` con quello trovato nel passaggio precedente)*.
3. Se ti viene chiesto di confermare l'identità del dispositivo, scrivi `yes` e premi Invio.
4. Inserisci la password di default: `raspberry`.
   *(Nota: mentre scrivi la password, non vedrai apparire caratteri a schermo per sicurezza)*.

## 5. Verifica del Dispositivo
Se ci sono più Raspberry Pi in rete e vuoi essere sicuro di essere connesso al tuo, puoi far lampeggiare il LED integrato:

- Lampeggiare con il ritmo "heartbeat" (battito cardiaco):
```bash
echo heartbeat | sudo tee /sys/class/leds/led0/trigger
```

- Lampeggiare regolarmente (timer):
```bash
echo timer | sudo tee /sys/class/leds/led0/trigger
```

- Spegnere il LED:
```bash
echo none | sudo tee /sys/class/leds/led0/trigger
```

- Ripristina il comportamento normale (attività disco):
```bash
echo mmc0 | sudo tee /sys/class/leds/led0/trigger
```

## 6. Configurazione Hostname

Rinomina il Raspberry Pi con il nome del tuo gruppo:

```bash
sudo hostnamectl set-hostname gruppo3ITT0
```
*(Sostituisci `gruppo3ITT0` con il nome assegnato al tuo gruppo)*.

### Verifica Finale
Controlla la configurazione di rete:
```bash
ifconfig
```

### Spegnimento e uscita
Per spegnere il Raspberry Pi in modo sicuro, esegui il comando:
```bash
sudo poweroff
```

Per uscire dalla sessione SSH, esegui il comando:
```bash
exit
```
